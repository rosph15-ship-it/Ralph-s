Example AI Prompt for Website Creation

Create a complete website in a single file called index.html.

Requirements:
- Use HTML, CSS, and JavaScript in one file.
- All styling must be inside a style tag.
- All scripts must be inside a script tag.
- The design must be modern, clean, and responsive.
- Use semantic HTML where possible.

Website Details:

Website Title:
[INSERT WEBSITE TITLE]

Purpose:
[What the website is for]

Theme / Style:
[Example: cyberpunk, neon, minimal, glassmorphism, modern tech]

Color Palette:
[Example: black background with neon green accents]

Layout Sections:

1. Header
   - Title
   - Optional navigation

2. Hero Section
   - Main headline
   - Subtitle
   - Call to action button

3. Main Content
   - Describe the content blocks you want

4. Cards / Features
   - Number of cards: X
   - Each card contains:
        - image
        - title
        - description
        - clickable link

5. Footer
   - copyright
   - optional links

Image Handling:
Images should be referenced locally in the same folder.

Example:
src="image.jpg"

Interactivity:
- button clicks
- navigation
- hover effects

Design Effects:
- smooth hover animations
- shadows or glow effects
- responsive layout

Output Format:
Return ONLY the final complete index.html file.
